## ▣ OVERVIEW

**Declassified Archive** is a zero-dependency, single-file web application that aggregates real, verified conspiracy theories, covert government operations, and declassified intelligence programs into a searchable, filterable intelligence dossier — styled like a classified government terminal.

Unlike fictional conspiracy databases, every entry is backed by:

- 🔴 Declassified government documents
- 🟡 U.S. Senate committee reports (Church Committee, Kerry Committee)
- 🟢 FOIA (Freedom of Information Act) releases
- 🔵 Verified journalism from major publications
- ⚪ Wikipedia's open-source knowledge base via live API

The credibility meter isn't a guess — it reflects the actual weight of documentary evidence.

## ▣ HOW THE CREDIBILITY SCORE WORKS

The score measures **documentary evidence**, not plausibility.

```
SCORE    BAR                   LABEL        MEANING
──────────────────────────────────────────────────────────────────
90–100%  ██████████  GREEN     CONFIRMED    Declassified docs, convictions, Senate reports
75–89%   ████████░░  GREEN     HIGH CONF.   Multiple credible sources, partial declassification  
50–74%   █████░░░░░  AMBER     MODERATE     Credible allegations, some corroboration
00–49%   ██░░░░░░░░  RED       DISPUTED     Limited evidence, contested, or debunked
```

**Signals used to infer score from Wikipedia text:**

| Signal Words | Effect |
|---|---|
| `confirmed` · `proven` · `declassified` · `admitted` · `convicted` | → pushes toward **90–100%** |
| `alleged` · `claimed` · `reportedly` · `disputed` · `unproven` | → pushes toward **45–70%** |
| `conspiracy theory` · `debunked` · `false` · `myth` | → pushes toward **20–50%** |

## ▣ ADDING NEW ENTRIES

Add a curated entry with verified sources to the `SEEDED` array in `the-dossier.html`:

```javascript
{
  id: "s13",
  title: "Your Operation Title",
  cat: "Government",   // Government | Military | Surveillance | Medical | Corporate
                       // Technology | Historical | Science | Political | Financial
  year: 1975,
  credibility: 88,     // 0–100
  wikiSlug: "Wikipedia_Page_Slug",
  seeded: true,
  desc: "Description of the operation or program...",
  sources: [
    {
      text: "Source Name — Publication or Agency (Year)",
      url: "https://verified-source-link.gov"
    }
  ],
  tags: ["TAG1", "TAG2", "TAG3"]
}
```

To expand the live Wikipedia feed, add slugs to `WIKI_QUEUE`:
```javascript
const WIKI_QUEUE = [
  // ...existing entries
  "Your_Wikipedia_Page_Slug"
];
```
